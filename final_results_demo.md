### 💾 核心算法实现

#### 1. 加权平均计算
```python
def weighted_avg(x, col, w_col='课时数'):
    """计算加权平均值"""
    w_sum = x[w_col].sum()
    if w_sum == 0:
        return 0
    return (x[col] * x[w_col]).sum() / w_sum
```

#### 2. 班级综合评分
```python
def calculate_class_score(row):
    """计算班级综合得分（0-100）"""
    # 出勤率权重40%
    attendance_score = row['出勤率'] * 0.4
    
    # 正确率权重40%
    correct_score = row['题目正确率'] * 0.4
    
    # 微课完成率权重20%
    micro_score = row['微课完成率'] * 0.2
    
    return (attendance_score + correct_score + micro_score) * 100
```
